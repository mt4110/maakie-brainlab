# S25-04 Observability Contract (v1)

Last Updated: 2026-02-26

## Scope

- `scripts/ops/current_point.py`
- `scripts/ops/s25_baseline_freeze.py`
- `scripts/ops/s25_obs_pr_summary.py`

## Log Level Contract

- Allowed levels: `OK`, `WARN`, `ERROR`, `SKIP`
- Any non-contract level is coerced to `WARN` with `invalid_level=<raw>` annotation.

## Storage Contract

- Root: `.local/obs/s25-ops/`
- Per-tool directory: `.local/obs/s25-ops/<tool>/`
- Per-run directory:
  - `<tool>__<UTCSTAMP>__<branch>__<sha7>`
  - Example: `current-point__20260226T163123Z__ops-s25-01-25-10__5d99cc9`

## Per-run Artifacts

- `run.meta.json`: run metadata and contract version
- `events.jsonl`: normalized event stream (`OK/WARN/ERROR/SKIP`)
- `summary.json`: counts and extra summary
- Tool-specific snapshot/log files

## PR Summary Artifacts

- `docs/evidence/s25-04/observability_latest.json`
- `docs/evidence/s25-04/observability_latest.md`

These are regenerated by:

- `make s25-baseline-freeze` (auto)
- `make s25-obs-summary` (manual refresh)
